# Kitchen Inventory and Management System

MINI-CAPSTONE PROJECT for STI
- Created as a mini capstone project for STI, this system allows for basic inventory tracking and student management for tracking inventory.

## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform
- Multiple interfaces for different types of users
- Tracks inventory

- Needs polishing
- Missing features
- Unnable to handle large amounts of data
